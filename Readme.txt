Για να εκτελεστεί η εφαρμογή απαιτεί την ύπαρξη της βιβλιοθήκης wxPython. Επίσης, για τη σωστή εμφάνιση των γραφικών συνίσταται η εκτέλεση από περιβάλλον Ubuntu Linux. Το username και το password που ζητούνται είναι τα αντίστοιχα της mysql, ενώ το administrator password είναι το "admin" και ορίζεται στην αρχή του αρχείου κώδικα.


Για εγκατάσταση της βιβλιοθήκης γραφικών wxPython για Ubuntu:

sudo apt-get install python-wxgtk2.8 python-wxtools wx2.8-doc wx2.8-examples wx2.8-headers wx2.8-i18n
